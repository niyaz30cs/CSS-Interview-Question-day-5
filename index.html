6.What are CSS combinator?

ANS:-

A combinator is the something that explainas the relationship between the selectors.

  • descendant selector(__space)
  • child selector(>)
  • adjacent sibiling selector(+)
  • general sibiling selector(~)

6.What are pseudo selectors in CSS?

ANS:-

A pseudo-class is used to define a special state of an element.

For example,it can be used to:

  • style an element when a user mouses over it.
  • style visited and unvisited links differntly.
  • style an element when it gets focus.
